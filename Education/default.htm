Lviv Open Data Bot (Education)

Bot is trained on this set.

Search engine is based on name field, so be precise when typing name, but feel free to make other part of your request more custom.

e.g invalid request: телефон українського католицького університету

valid request: де знаходиться український католицький університет

As you see, there are two properties you can ask for - phone number and address - all entities contain these ones.
Basically, you get all information available, but you can specify that you want only phone number or e.g. address.
